# 🖥️ واجهة برمجة التطبيقات للمنشأة (Backend API)

هذا هو خادم الـ API المبني باستخدام Node.js و Express و TypeScript لإدارة الموارد البشرية والمهام، ومزود بـ Gemini AI لتقديم ملخصات وتقييمات ذكية.

## 🛠️ المتطلبات التشغيلية
- **Node.js**: إصدار 18 أو أحدث.
- **npm** أو **yarn** أو **bun** لإدارة الحزم.

## 🚀 التشغيل والتثبيت المحلي

1. **تثبيت المكتبات:**
   ```bash
   npm install
   ```

2. **تهيئة البيئة:**
   قم بنسخ ملف `.env.example` إلى ملف جديد باسم `.env` وعيّن القيم المطلوبة:
   ```bash
   cp .env.example .env
   ```
   تأكد من ضبط `GEMINI_API_KEY` و `JWT_SECRET`.

3. **تشغيل الخادم في وضع التطوير (Development):**
   ```bash
   npm run dev
   ```
   سيبدأ الخادم بالعمل بشكل مستقل على المنفذ `5000` (أو المنفذ المذكور في ملف الـ `.env`).

4. **بناء المشروع للإنتاج (Production Build):**
   ```bash
   npm run build
   ```

5. **تشغيل نسخة الإنتاج:**
   ```bash
   npm start
   ```

## 📂 الهيكل العام للقرص
- `server.ts`: نقطة الدخول والمتحكمات البرمجية للمسارات.
- `src/types.ts`: أنواع البيانات المشتركة والنماذج البرمجية.
- `data/db.json`: قاعدة البيانات المحلية الخفيفة (JSON) لتخزين بيانات الموظفين، المهام، الحضور، الإعلانات، والتقييمات.
- `uploads/`: المجلد المحلي لحفظ المستندات المرفوعة من الكادر.

## 🔒 حماية وتأمين البيانات
- جميع المسارات محمية بالكامل باستخدام مسار تحقق JWT الـ `Bearer` باستثناء بوابة تسجيل الدخول.
- يتم تشفير وحفظ كلمات مرور الموظفين باستخدام خوارزميات التجزئة SHA256 والتمليح (Salting) لضمان السرية المطلقة.
